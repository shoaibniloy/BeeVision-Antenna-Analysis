# BeeVision-A-Two-Stage-Pipeline-for-Real-Time-Antenna-Tracking-and-Behavior-Analysis-in-Honeybees
BeeVision is a morphology-guided two-stage pipeline for real-time honeybee antenna tracking. It replaces learned features with signal-based detection and optical flow, boosting accuracy to 92.7% and enabling behavioral metrics like lateralization and trophallaxis for early colony health assessment.
